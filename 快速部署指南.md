# 快速部署指南 🚀

3种最简单的部署方式，选一个即可！

---

## 🎯 方案1：Vercel（推荐，最简单）

### 为什么选择 Vercel？
- ✅ 完全免费
- ✅ 3分钟部署
- ✅ 自动 HTTPS
- ✅ 全球 CDN

### 部署步骤

#### 1. 推送代码到 GitHub

```bash
# 如果还没有 Git 仓库
git init
git add .
git commit -m "准备部署"

# 在 GitHub 上创建新仓库，然后：
git remote add origin https://github.com/你的用户名/outfit-simulator.git
git branch -M main
git push -u origin main
```

#### 2. 部署到 Vercel

**方式A：通过网页（最简单）**

1. 访问：https://vercel.com
2. 用 GitHub 账号登录
3. 点击 "Add New..." → "Project"
4. 选择你的 GitHub 仓库
5. 点击 "Deploy"
6. 等待2-3分钟
7. 完成！得到一个 URL

**方式B：通过命令行**

```bash
# 安装 Vercel CLI
npm install -g vercel

# 登录
vercel login

# 部署
vercel

# 按照提示操作即可
```

#### 3. 访问你的网站

部署完成后，Vercel 会给你一个 URL，例如：
```
https://outfit-simulator.vercel.app
```

## 🚂 方案2：Railway（超级简单）

### 为什么选择 Railway？
- ✅ 完全免费（有额度）
- ✅ 1分钟部署
- ✅ 自动检测配置

### 部署步骤

#### 2. 部署到 Railway

1. 访问：https://railway.app
2. 用 GitHub 账号登录
3. 点击 "New Project"
4. 选择 "Deploy from GitHub repo"
5. 选择你的仓库
6. Railway 自动检测并部署
7. 完成！

#### 3. 访问你的网站

Railway 会自动生成一个 URL。

## 🎨 方案3：Render（完全免费）

### 为什么选择 Render？
- ✅ 完全免费
- ✅ 简单易用
- ✅ 自动 HTTPS

### 部署步骤

#### 1. 推送代码到 GitHub（同上）

#### 2. 部署到 Render

1. 访问：https://render.com
2. 注册账号
3. 点击 "New +" → "Web Service"
4. 连接 GitHub 仓库
5. 配置：
   - **Name**: outfit-simulator
   - **Build Command**: `pip install -r requirements.txt`
   - **Start Command**: `python run.py`
6. 点击 "Create Web Service"
7. 等待5-10分钟
8. 完成！

#### 3. 访问你的网站

Render 会给你一个 URL。

## ⚠️ 重要提示

### 1. 使用正确的 requirements.txt

对于 Vercel，使用：
```bash
cp requirements-vercel.txt requirements.txt
```

或者确保 `requirements.txt` 中使用 `opencv-python-headless`：
```
opencv-python-headless>=4.8.0  # 不是 opencv-python
```

### 2. 模型文件

确保 `models/best_model.h5` 存在且小于 50MB。

### 3. 环境变量

如果需要，在平台设置中添加环境变量。

## 🐛 常见问题

### Q1: 部署失败怎么办？

**检查**：
1. `requirements.txt` 是否正确
2. 查看部署日志
3. 确保代码在本地能运行

### Q2: 网站打不开？

**检查**：
1. 部署是否完成
2. URL 是否正确
3. 查看错误日志

### Q3: 图片上传失败？

**原因**：
- 免费版有文件大小限制
- 可能需要使用外部存储（如 Cloudinary）
