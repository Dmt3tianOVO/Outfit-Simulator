# 立即部署 - 3分钟上线 🚀

你的代码已经在 GitHub 上了，现在只需要 3 分钟就能上线！

---

## ⚠️ 关于 Cloudflare

**Cloudflare Pages 不支持 Python Flask 应用**

但是，**Vercel 和 Railway 比 Cloudflare 更适合你的项目**！

- ✅ 同样快速
- ✅ 同样有全球 CDN
- ✅ 同样免费
- ✅ 支持 Python

---

## 🎯 最简单的方式：Railway（1分钟）

### 步骤1：打开 Railway

在浏览器中访问：**https://railway.app**

### 步骤2：登录

点击 **"Login"** → 选择 **"Login with GitHub"**

### 步骤3：创建项目

点击 **"New Project"**

### 步骤4：选择仓库

点击 **"Deploy from GitHub repo"**

在列表中找到你的仓库（outfit-simulator），点击它

### 步骤5：等待部署

Railway 会自动：
- 检测 Python 项目
- 安装依赖
- 启动应用

等待 2-3 分钟...

### 步骤6：获取 URL

部署完成后：
1. 点击你的项目
2. 点击 "Settings"
3. 找到 "Domains"
4. 点击 "Generate Domain"
5. 获得一个 URL，例如：`https://outfit-simulator-production.up.railway.app`

### 步骤7：访问网站

点击 URL，你的网站就上线了！🎉

---

## 🚀 或者使用 Vercel（3分钟）

### 步骤1：打开 Vercel

在浏览器中访问：**https://vercel.com**

### 步骤2：登录

点击 **"Sign Up"** → 选择 **"Continue with GitHub"**

### 步骤3：导入项目

1. 点击 **"Add New..."**
2. 选择 **"Project"**
3. 在 "Import Git Repository" 中找到你的仓库
4. 点击 **"Import"**

### 步骤4：配置（自动）

Vercel 会自动检测配置，直接点击 **"Deploy"**

### 步骤5：等待部署

等待 2-3 分钟...

### 步骤6：完成

部署完成后，你会看到：
- ✅ Congratulations!
- 🔗 你的网站 URL

点击 URL 访问！

---

## ⚠️ 如果 Vercel 部署失败

### 修复依赖问题

在你的本地项目中运行：

```bash
# 使用 Vercel 专用依赖
cp requirements-vercel.txt requirements.txt

# 提交更改
git add requirements.txt
git commit -m "使用 Vercel 依赖"
git push
```

然后在 Vercel 中点击 **"Redeploy"**

## 🐛 常见问题

### Q: 为什么不能用 Cloudflare Pages？

**A**: Cloudflare Pages 只支持静态网站和 JavaScript 框架，不支持 Python Flask。

### Q: Railway 和 Vercel 哪个更好？

**A**: 
- **Railway**: 更简单，支持持久化存储
- **Vercel**: 更快，但历史记录会在重启后丢失

### Q: 部署需要多长时间？

**A**: 
- Railway: 2-3 分钟
- Vercel: 2-3 分钟

### Q: 免费吗？

**A**: 是的，两个平台都有免费额度，足够个人项目使用。

### Q: 可以使用自己的域名吗？

**A**: 可以，在平台设置中添加自定义域名。
